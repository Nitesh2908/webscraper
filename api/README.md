Install python 3.12

Clone Repo
- git clone <lululemon repo>

Go to Lululemon directory
- cd Lululemon

Activate virtual env
- .venv/Scripts/activate

Install requirements.txt
- pip install requirements.txt

Run Flask application
- python api/main.py

Call the get api
- http://localhost:8080/products/

To run test cases
- pytest api/tests
